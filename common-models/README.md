# common-models

Proyecto JAR para compartir modelos DTO entre microservicios.

## Instalar en Maven local

```bash
mvn clean install
```

## Usar en otro microservicio

```xml
<dependency>
    <groupId>com.mitocode.microservices</groupId>
    <artifactId>common-models</artifactId>
    <version>0.0.1-SNAPSHOT</version>
</dependency>
```
